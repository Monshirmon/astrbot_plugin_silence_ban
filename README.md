# 说不理你就不理你（SilenceBan）

AstrBot 黑名单插件。当 LLM 回复 "不理你了" / "真的不理你了" 达到设定次数后，自动将用户加入黑名单。支持随机时长封禁，管理员可在插件界面配置随机时长区间，也可手动封禁/解封用户。

## 功能

- **话术自动封禁**：检测 LLM 回复中的 "不理你了" / "真的不理你了"，累计达到设定次数后自动以随机时长封禁用户
- **随机时长封禁**：封禁时长在插件配置的区间内随机生成，管理员也可通过 `/random-ban` 命令手动触发
- **完整手动管理**：保留 `/ban` `/pass` `/dec-ban` `/ban-reset` 等全套管理命令
- **话术计数器**：`/silence-counter` 查看各用户触发次数

## 配置

在 AstrBot WebUI 插件管理界面可配置以下选项：

| 配置项 | 说明 | 默认值 |
|---|---|---|
| `enable` | 是否启用封禁功能 | true |
| `cache_ttl` | 缓存存活时间（秒） | 60 |
| `auto_ban_enable` | 是否启用话术自动封禁 | true |
| `trigger_count` | 触发多少次话术后自动封禁 | 2 |
| `min_random_ban_seconds` | 随机封禁最短时长（秒） | 60 |
| `max_random_ban_seconds` | 随机封禁最长时长（秒） | 86400（1天） |

## 命令列表

### 封禁命令

| 命令 | 格式 | 说明 |
|---|---|---|
| `/ban` | `/ban <@用户\|UID> [时间] [理由] [UMO]` | 会话内封禁用户 |
| `/ban-all` | `/ban-all <@用户\|UID> [时间] [理由]` | 全局封禁用户 |
| `/random-ban` | `/random-ban <@用户\|UID> [UMO]` | 以随机时长封禁用户 |
| `/dec-ban` | `/dec-ban <@用户\|UID> [时间] [理由] [UMO]` | 减少会话封禁时限 |
| `/dec-ban-all` | `/dec-ban-all <@用户\|UID> [时间] [理由]` | 减少全局封禁时限 |

### 解限命令

| 命令 | 格式 | 说明 |
|---|---|---|
| `/pass` | `/pass <@用户\|UID> [时间] [理由] [UMO]` | 会话内解限用户 |
| `/pass-all` | `/pass-all <@用户\|UID> [时间] [理由]` | 全局解限用户 |
| `/dec-pass` | `/dec-pass <@用户\|UID> [时间] [理由] [UMO]` | 减少会话解限时限 |
| `/dec-pass-all` | `/dec-pass-all <@用户\|UID> [时间] [理由]` | 减少全局解限时限 |
| `/ban-reset` | `/ban-reset <@用户\|UID>` | 清除用户所有封禁/解限记录 |

### 查询与控制

| 命令 | 说明 |
|---|---|
| `/banlist` | 查看当前封禁/解限名单 |
| `/silence-counter` | 查看话术计数器状态 |
| `/ban-enable` | 启用封禁功能 |
| `/ban-disable` | 停用封禁功能 |
| `/ban-help` | 查看帮助信息 |

> 以上封禁/解限命令仅管理员可用。

## 时间格式

数字 + 单位，如 `1d`（1天）、`2h`（2小时）、`30m`（30分钟）、`10s`（10秒）。不填或填 `0` 表示永久。

## 自动封禁工作流程

1. LLM 回复中包含 "不理你了" 或 "真的不理你了"
2. 该用户在对应会话的话术计数器 +1
3. 未达阈值时，在 LLM 回复末尾追加警告提示
4. 达到阈值（默认2次）后，自动封禁该用户随机时长，并追加封禁通知
5. 计数器归零

- msgpack

## 数据存储

所有数据存储在 AstrBot 的 `data` 目录下：

| 文件 | 说明 |
|---|---|
| `ban_list.json` | 会话级封禁名单 |
| `banall_list.json` | 全局封禁名单 |
| `pass_list.json` | 会话级解限名单 |
| `passall_list.json` | 全局解限名单 |
| `umo_ban_list.json` | 会话级封禁 |
| `umo_pass_list.json` | 会话级解限 |
| `silence_counter.json` | 话术计数器 |
